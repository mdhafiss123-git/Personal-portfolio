## 📝 Customization

### Change Title
Edit in `index.html`:
```html
<span class="title-line">Your Title &</span>
<span class="title-highlight">Your Subtitle</span>
```

### Change Statistics
```html
<div class="stat-number-large">YOUR NUMBER</div>
<div class="stat-label-large">YOUR LABEL</div>
```

### Change Colors
Edit in `assets/css/styles.css`:
```css
.title-highlight {
    background: linear-gradient(135deg, #yourcolor1, #yourcolor2, #yourcolor3);
}
```

### Change Button Text
```html
<span class="btn-icon">YOUR_EMOJI</span> Button Text
```
